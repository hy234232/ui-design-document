화살표 색상
사용 방법
• 두 프레임을 순서대로 선택하세요 (첫 번째 = 출발 · 두 번째 = 도착).
• 자동 생성 ON 상태에서 2개 프레임이 선택되면 즉시 화살표가 그려집니다.
• 자동 OFF면 위 "선택한 두 프레임 연결" 버튼으로 수동 생성합니다.
• 화살표는 ELBOW 라인(직각 라운드), 시작점은 원형(●), 끝점은 화살표(→).

프레임 사이즈
체크한 타이틀명마다 타이틀 헤더와 빈 프레임을 캔버스에 생성합니다.
여러 프레임을 선택한 뒤 실행하면 각 프레임 아래에 프레임명 기준 최소/최대 사이즈 라벨을 생성합니다.
